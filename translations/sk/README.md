## 🌱 Začnite

Tento kurz obsahuje lekcie pokrývajúce základy tvorby AI Agentov. Každá lekcia sa venuje svojej téme, takže začnite tam, kde chcete!

Tento kurz má podporu viacerých jazykov. Prejdite na naše [dostupné jazyky tu](#-multi-language-support).

Ak je to váš prvýkrát, čo pracujete s modelmi generatívnej AI, pozrite si náš kurz [Generatívna AI pre Začiatočníkov](https://aka.ms/genai-beginners), ktorý zahŕňa 21 lekcií o tvorbe s GenAI.

Nezabudnite [dať hviezdičku (🌟) tomuto repozitáru](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) a [forknúť tento repozitár](https://github.com/microsoft/ai-agents-for-beginners/fork), aby ste mohli spúšťať kód.

### Spoznajte ďalších študentov, získajte odpovede na svoje otázky

Ak máte nejaké otázky alebo máte problém pri tvorbe AI Agentov, pripojte sa k nášmu vyhradenému kanálu na Discord v [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Čo potrebujete

Každá lekcia v tomto kurze obsahuje príklady kódu, ktoré nájdete v priečinku code_samples. Môžete si [forknúť tento repozitár](https://github.com/microsoft/ai-agents-for-beginners/fork) a vytvoriť si vlastnú kópiu.

Príklady kódu v týchto cvičeniach používajú Microsoft Agent Framework s Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - vyžaduje sa Azure účet

Tento kurz používa nasledujúce AI Agent frameworky a služby od Microsoftu:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Niektoré príklady kódu tiež podporujú alternatívnych poskytovateľov kompatibilných s OpenAI, ako je [MiniMax](https://platform.minimaxi.com/), ktorý ponúka modely s veľkým kontextom (až 204K tokenov). Pozrite si [nastavenie kurzu](./00-course-setup/README.md) pre podrobnosti o konfigurácii.

Pre viac informácií o spustení kódu pre tento kurz prejdite na [Nastavenie kurzu](./00-course-setup/README.md).
