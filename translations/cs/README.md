## 🌱 Začínáme

Tento kurz obsahuje lekce pokrývající základy vytváření AI Agentů. Každá lekce se věnuje svému tématu, začněte tedy tam, kde chcete!

Kurz má vícejazyčnou podporu. Navštivte naše [dostupné jazyky zde](#-multi-language-support). 

Pokud sestavujete s modely generativní AI poprvé, podívejte se na náš kurz [Generative AI For Beginners](https://aka.ms/genai-beginners), který obsahuje 21 lekcí o tvorbě s GenAI.

Nezapomeňte [ohvězdičkovat (🌟) tento repozitář](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) a [vytvořit fork tohoto repozitáře](https://github.com/microsoft/ai-agents-for-beginners/fork) pro spuštění kódu.

### Poznejte ostatní studenty, získejte odpovědi na své otázky

Pokud se zaseknete nebo budete mít otázky ohledně tvorby AI Agentů, připojte se na náš specializovaný Discord kanál v [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Co potřebujete

Každá lekce v tomto kurzu zahrnuje ukázky kódu, které najdete ve složce code_samples. Můžete si [vytvořit fork tohoto repozitáře](https://github.com/microsoft/ai-agents-for-beginners/fork) a vytvořit si vlastní kopii.  

Ukázky kódu v těchto cvičeních používají Microsoft Agent Framework s Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Vyžaduje Azure účet

Tento kurz využívá následující AI Agent frameworky a služby od Microsoftu:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Některé ukázky kódu také podporují alternativní poskytovatele kompatibilní s OpenAI, například [MiniMax](https://platform.minimaxi.com/), který nabízí velké kontextové modely (až 204 tisíc tokenů). Podívejte se na [Nastavení kurzu](./00-course-setup/README.md) pro podrobnosti o konfiguraci.

Pro více informací o spuštění kódu k tomuto kurzu přejděte na [Nastavení kurzu](./00-course-setup/README.md).
