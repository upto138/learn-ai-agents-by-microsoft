>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> Це дасть вам все необхідне для проходження курсу з набагато швидшим завантаженням.

## 🌱 Початок роботи

Цей курс охоплює уроки, що розглядають основи створення AI Агентів. Кожен урок присвячено окремій темі, тому починайте з будь-якого, який вам подобається!

Для цього курсу є підтримка багатьох мов. Перейдіть до наших [доступних мов тут](#-multi-language-support).

Якщо ви вперше створюєте з моделями Генеративного ШІ, подивіться наш курс [Генеративний ШІ для Початківців](https://aka.ms/genai-beginners), який містить 21 урок зі створення з GenAI.

Не забудьте [поставити ⭐ цьому репозиторію](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) і [зробити форк цього репозиторію](https://github.com/microsoft/ai-agents-for-beginners/fork), щоб запускати код.

### Познайомтеся з іншими учнями, отримайте відповіді на свої питання

Якщо у вас виникли труднощі або є питання щодо створення AI Агентів, приєднуйтесь до нашого спеціального Discord-каналу в [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Що вам потрібно

Кожен урок у цьому курсі містить приклади коду, які можна знайти в папці code_samples. Ви можете [зробити форк цього репозиторію](https://github.com/microsoft/ai-agents-for-beginners/fork), щоб створити власну копію.

Приклади коду в цих вправ використовують Microsoft Agent Framework з Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Треба обліковий запис Azure

У цьому курсі використовуються такі AI Agent фреймворки та сервіси від Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Деякі приклади коду також підтримують альтернативних провайдерів, сумісних із OpenAI, таких як [MiniMax](https://platform.minimaxi.com/), який пропонує моделі з великим контекстом (до 204K токенів). Дивіться [Налаштування курсу](./00-course-setup/README.md) для деталей конфігурації.

Для отримання додаткової інформації про запуск коду цього курсу перейдіть до [Налаштування курсу](./00-course-setup/README.md).
