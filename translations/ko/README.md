> ```

## 🌱 시작하기

이 강좌는 AI 에이전트 구축의 기본을 다루는 강의들로 구성되어 있습니다. 각 강의는 독립된 주제를 다루므로 원하는 곳에서 시작하세요!

이 강좌는 다국어 지원이 됩니다. 지원되는 언어는 [여기](#-multi-language-support)에서 확인하세요.

제너레이티브 AI 모델로 빌드를 처음 시작하는 경우, 21개의 GenAI 빌드 관련 강의가 포함된 [초보자를 위한 생성 AI 강좌](https://aka.ms/genai-beginners)를 참고하세요.

코드를 실행하려면 이 저장소를 [별(🌟) 표시](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst)하고 [포크](https://github.com/microsoft/ai-agents-for-beginners/fork)하는 것을 잊지 마세요.

### 다른 학습자 만나기, 질문하기

AI 에이전트 구축에 대해 질문이 있거나 막혔을 때는 [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)의 전용 Discord 채널에 참여하세요.

### 필요한 것

강의마다 코드 예제가 포함되어 있으며, 코드는 code_samples 폴더에서 찾을 수 있습니다. [이 저장소를 포크](https://github.com/microsoft/ai-agents-for-beginners/fork)하여 자신만의 복사본을 만드세요.

이 예제 코드들은 Microsoft Agent Framework 와 Azure AI Foundry Agent Service V2를 활용합니다:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure 계정 필요

이 강좌에서 사용하는 Microsoft AI 에이전트 프레임워크 및 서비스:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

일부 코드 예제는 [MiniMax](https://platform.minimaxi.com/) 같은 OpenAI 호환 대체 공급자도 지원하며, 최대 204K 토큰의 대용량 컨텍스트 모델을 제공합니다. 설정 관련 자세한 내용은 [강좌 설정](./00-course-setup/README.md)을 확인하세요.

강좌용 코드를 실행하는 방법에 대한 자세한 내용은 [강좌 설정](./00-course-setup/README.md)을 참고하세요.
