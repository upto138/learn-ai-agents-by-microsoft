## 🌱 Alustamine

See kursus sisaldab õppetunde, mis katavad AI agentide ehituse põhialuseid. Iga õppetund käsitleb oma teemat, nii et alusta sellest, mis sulle meeldib!

Sellel kursusel on mitmekeelne tugi. Mine meie [saadaval olevate keelte juurde siia](#-mitmekeelne-tugi).

Kui see on sinu esimene kord Generatiivse AI mudelitega ehitades, vaata meie [Generative AI For Beginners](https://aka.ms/genai-beginners) kursust, mis sisaldab 21 õppetundi GenAI kasutamisest.

Ära unusta [märgista seda hoidlat (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ja [loo sellest hoidlast oma fork](https://github.com/microsoft/ai-agents-for-beginners/fork), et käivitada kood.

### Kohtu teiste õppijatega, saa vastuseid oma küsimustele

Kui jääd hätta või sul on küsimusi AI agentide loomise kohta, ühine meie pühendatud Discord kanaliga [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Mida vajad

Igas õppetunnis on koodinäited, mis asuvad code_samples kaustas. Sa võid [forkida selle hoidla](https://github.com/microsoft/ai-agents-for-beginners/fork), et teha oma koopiat.

Neis harjutustes kasutatud koodinäited kasutavad Microsoft Agent Frameworki koos Azure AI Foundry Agent Service V2-ga:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - nõuab Azure kontot

See kursus kasutab Microsofti järgmisi AI agentide raamistikuid ja teenuseid:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Mõned koodinäited toetavad ka OpenAI-le vastavaid alternatiivseid pakkujaid nagu [MiniMax](https://platform.minimaxi.com/), mis pakub suure konteksti mudeleid (kuni 204K tokenit). Vaata üksikasju konfiguratsiooni kohta [Course Setup](./00-course-setup/README.md) failist.

Lisateavet selle kursuse koodi käivitamise kohta leiad failist [Course Setup](./00-course-setup/README.md).
