>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> Dette gir deg alt du trenger for å fullføre kurset med mye raskere nedlasting.

## 🌱 Komme i gang

Dette kurset har leksjoner som dekker grunnleggende om å bygge AI-agenter. Hver leksjon dekker sitt eget tema, så start hvor du vil!

Det finnes flerspråklig støtte for dette kurset. Se våre [tilgjengelige språk her](#-multi-language-support).

Hvis dette er første gang du bygger med generative AI-modeller, sjekk ut vårt kurs [Generative AI For Beginners](https://aka.ms/genai-beginners), som inkluderer 21 leksjoner om bygging med GenAI.

Ikke glem å [stjerne (🌟) dette depotet](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) og [fork dette depotet](https://github.com/microsoft/ai-agents-for-beginners/fork) for å kjøre koden.

### Møt andre elever, få svar på spørsmålene dine

Hvis du står fast eller har spørsmål om å bygge AI-agenter, bli med i vår dedikerte Discord-kanal i [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Hva du trenger

Hver leksjon i dette kurset inkluderer kodeeksempler, som finnes i mappen code_samples. Du kan [forke dette depotet](https://github.com/microsoft/ai-agents-for-beginners/fork) for å lage din egen kopi.

Kodeeksemplene i disse øvelsene bruker Microsoft Agent Framework med Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure-konto kreves

Dette kurset bruker følgende AI-agent-rammeverk og tjenester fra Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Noen kodeeksempler støtter også alternative OpenAI-kompatible tilbydere som [MiniMax](https://platform.minimaxi.com/), som tilbyr modeller med stor kontekst (opptil 204K tokens). Se [Course Setup](./00-course-setup/README.md) for konfigurasjonsdetaljer.

For mer informasjon om hvordan du kjører koden for dette kurset, gå til [Course Setup](./00-course-setup/README.md).
