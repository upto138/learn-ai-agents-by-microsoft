> **স্থানীয়ভাবে ক্লোন করতে চাচ্ছেন?**
>
> এই রিপোজিটরিতে ৫০+ ভাষার অনুবাদ অন্তর্ভুক্ত রয়েছে যা ডাউনলোড সাইজ অনেক বাড়িয়ে তোলে। অনুবাদ ছাড়া ক্লোন করতে sparse checkout ব্যবহার করুন:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (উইন্ডোজ):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> এতে দ্রুত ডাউনলোড হবে এবং কোর্স সম্পন্ন করার জন্য যা যা দরকার সবই পেয়ে যাবেন।

## 🌱 শুরু করা হচ্ছে

এই কোর্সে AI এজেন্ট তৈরি করার মৌলিক বিষয়গুলোcovered করা হয়েছে। প্রতিটি লেসনে তার নিজস্ব বিষয় আছে তাই যেখান থেকে খুশি সেখান থেকে শুরু করুন!

এই কোর্সটির জন্য বহু-ভাষার সমর্থন দেওয়া হয়েছে। আমাদের [উপলব্ধ ভাষাগুলো এখানে](#-multi-language-support) দেখুন।

আপনি যদি প্রথমবার জেনারেটিভ AI মডেল ব্যবহার করে তৈরি করছেন, তাহলে আমাদের [Generative AI For Beginners](https://aka.ms/genai-beginners) কোর্সটি দেখুন, যেখানে GenAI দিয়ে তৈরি করার ২১টি লেসন রয়েছে।

এই রিপোজিটরিটি [star (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) এবং [fork করুন](https://github.com/microsoft/ai-agents-for-beginners/fork) কোড চালানোর জন্য।

### অন্যান্য শিক্ষার্থীদের সাথে পরিচিত হন, প্রশ্নের উত্তর পান

যদি আটকে যান বা AI এজেন্ট তৈরি সংক্রান্ত কোনো প্রশ্ন থাকে, আমাদের বিশেষ Discord চ্যানেলে যোগ দিন [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)।

### আপনার প্রয়োজনীয়তা

এই কোর্সের প্রতিটি লেসনে কোড উদাহরণ রয়েছে, যা code_samples ফোল্ডারে পাওয়া যাবে। আপনি [এই রিপো ফর্ক](https://github.com/microsoft/ai-agents-for-beginners/fork) করে নিজের কপি তৈরি করতে পারেন।

এই কোড উদাহরণগুলো Microsoft Agent Framework যেন Azure AI Foundry Agent Service V2 এর সাথে কাজ করে:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure একাউন্ট প্রয়োজন

এই কোর্সে Microsoft থেকে নিম্নলিখিত AI এজেন্ট ফ্রেমওয়ার্ক এবং পরিষেবাগুলো ব্যবহৃত হয়েছে:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

কিছু কোড নমুনা বিকল্প OpenAI-সঙ্গতিপূর্ণ প্রদানকারী যেমন [MiniMax](https://platform.minimaxi.com/)কেও সমর্থন করে, যা বড়-কন্টেক্সট মডেল (২৪০ হাজার টোকেন পর্যন্ত) অফার করে। কনফিগারেশন বিস্তারিত জন্য দেখুন [Course Setup](./00-course-setup/README.md)।

এই কোর্সের কোড রান করার আরও তথ্যের জন্য দেখুন [Course Setup](./00-course-setup/README.md)।
