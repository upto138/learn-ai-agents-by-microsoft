## 🌱 Почетак

Овај курс има лекције које покривају основе израде AI агената. Свака лекција се бави својом темом, па почните где год желите!

Постоји подршка за више језика за овај курс. Погледајте наше [доступне језике овде](#-multi-language-support).

Ако је ово ваш први пут да правите уз помоћ генеративних AI модела, погледајте наш курс [Generative AI For Beginners](https://aka.ms/genai-beginners), који укључује 21 лекцију за рад са GenAI.

Не заборавите да [означите (🌟) овај репо](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) и [форкујете овај репо](https://github.com/microsoft/ai-agents-for-beginners/fork) да бисте покренули код.

### Упознајте друге ученике, добијте одговоре на питања

Ако запнете или имате питања о изради AI агената, придружите се нашем посебном Discord каналу у [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Шта вам је потребно

Свакa лекцијa у овом курсу укључује примере кода који се налазе у фасцикли code_samples. Можете [форковати овај репо](https://github.com/microsoft/ai-agents-for-beginners/fork) да направите своју копију.

Примери кода у овим вежбама користе Microsoft Agent Framework са Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - потребан Azure налог

Овај курс користи следеће AI Agent фрејмворке и сервисе из Microsoft-а:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Неки примери кода такође подржавају алтернативне провајдере компатибилне са OpenAI као што је [MiniMax](https://platform.minimaxi.com/), који нуди моделе великог контекста (до 204К токена). Погледајте [постављање курса](./00-course-setup/README.md) за детаље о конфигурацији.

За више информација о покретању кода за овај курс идите на [постављање курса](./00-course-setup/README.md).
