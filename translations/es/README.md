> **¿Prefieres Clonar Localmente?**
>
> Este repositorio incluye traducciones a más de 50 idiomas, lo que incrementa significativamente el tamaño de la descarga. Para clonar sin traducciones, usa el checkout disperso:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> Esto te da todo lo que necesitas para completar el curso con una descarga mucho más rápida.

## 🌱 Comenzando

Este curso tiene lecciones que cubren los fundamentos para construir Agentes de IA. Cada lección aborda su propio tema, ¡así que empieza donde quieras!

Hay soporte multilingüe para este curso. Ve nuestros [idiomas disponibles aquí](#-multi-language-support).

Si es la primera vez que construyes con modelos de IA Generativa, consulta nuestro curso [IA Generativa para Principiantes](https://aka.ms/genai-beginners), que incluye 21 lecciones sobre construcción con GenAI.

No olvides [darle una estrella (🌟) a este repositorio](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) y [hacer fork a este repositorio](https://github.com/microsoft/ai-agents-for-beginners/fork) para ejecutar el código.

### Conoce a Otros Estudiantes, Resuelve Tus Dudas

Si te atascas o tienes preguntas sobre la construcción de Agentes de IA, únete a nuestro canal dedicado en Discord en el [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Lo Que Necesitas

Cada lección en este curso incluye ejemplos de código, que puedes encontrar en la carpeta code_samples. Puedes [hacer fork a este repositorio](https://github.com/microsoft/ai-agents-for-beginners/fork) para crear tu propia copia.

Los ejemplos de código en estos ejercicios utilizan Microsoft Agent Framework con Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Se requiere cuenta de Azure

Este curso utiliza los siguientes frameworks y servicios de Agentes de IA de Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Algunos ejemplos de código también soportan proveedores alternativos compatibles con OpenAI como [MiniMax](https://platform.minimaxi.com/), que ofrece modelos de contexto amplio (hasta 204K tokens). Consulta el [Configuración del Curso](./00-course-setup/README.md) para detalles de configuración.

Para más información sobre cómo ejecutar el código de este curso, visita el [Configuración del Curso](./00-course-setup/README.md).
