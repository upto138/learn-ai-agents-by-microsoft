## 🌱 شروع کرنا

یہ کورس AI ایجنٹس بنانے کے بنیادی اصولوں کو کور کرتا ہے۔ ہر سبق اپنی موضوعی تحریر رکھتا ہے، لہٰذا جہاں چاہیں وہاں سے شروع کریں!

اس کورس کے لیے ملٹی لنگویج سپورٹ موجود ہے۔ ہمارے [دستیاب زبانوں یہاں](#-multi-language-support) جائیں۔ 

اگر آپ پہلی بار جنریٹو AI ماڈلز کے ساتھ کام کر رہے ہیں، تو ہمارا [جنریٹو AI برائے ابتدائی](https://aka.ms/genai-beginners) کورس دیکھیں، جس میں GenAI کے ساتھ بنانے کے 21 اسباق شامل ہیں۔

اس ریپوزٹری کو [ستارہ (🌟) دینا](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) اور [فورک کرنا](https://github.com/microsoft/ai-agents-for-beginners/fork) نا بھولیں تاکہ کوڈ چلایا جا سکے۔

### دوسرے سیکھنے والوں سے ملیں، اپنے سوالات کے جواب حاصل کریں

اگر آپ کسی مسئلے میں پھنس جائیں یا AI ایجنٹس بنانے کے بارے میں کوئی سوال ہو، تو ہمارے مخصوص Discord چینل میں شامل ہوں [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)۔

### آپ کو کیا چاہیے

اس کورس کے ہر سبق میں کوڈ کی مثالیں شامل ہیں، جو code_samples فولڈر میں ملتی ہیں۔ آپ اپنی کاپی بنانے کے لیے [اس ریپوزٹری کو فورک کر سکتے ہیں](https://github.com/microsoft/ai-agents-for-beginners/fork)۔  

ان مشقوں میں دی گئی کوڈ مثالیں Microsoft Agent Framework کے ساتھ Azure AI Foundry Agent Service V2 استعمال کرتی ہیں:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure اکاؤنٹ ضروری ہے

یہ کورس Microsoft کے درج ذیل AI ایجنٹ فریم ورکس اور خدمات استعمال کرتا ہے:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

کچھ کوڈ نمونے OpenAI-مطابق دیگر فراہم کنندگان جیسے [MiniMax](https://platform.minimaxi.com/) کی حمایت بھی کرتے ہیں، جو بڑے کانٹیکسٹ ماڈلز (204K ٹوکن تک) پیش کرتا ہے۔ کنفیگریشن کی تفصیلات کے لیے [کورس سیٹ اپ](./00-course-setup/README.md) دیکھیں۔

اس کورس کے کوڈ چلانے کے بارے میں مزید معلومات کے لیے، [کورس سیٹ اپ](./00-course-setup/README.md) پر جائیں۔
